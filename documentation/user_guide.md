# User Guide

The active v2 app is a frontend-only storefront prototype.

## Casual User Flow

1. Use the simulated log in or sign up screen.
2. Open the Nearby tab to browse food storefronts inside the mocked 2KM area.
3. Open a storefront to view its food catalog.
4. Subscribe to join the storefront thread.
5. Unsubscribe from storefronts you no longer want to follow.
6. Use Account to change display name, switch user type, or logout.

## Storefront Owner Flow

1. Use Account to switch to Owner.
2. Open the Storefronts tab.
3. Create a storefront or manage the existing mock storefront.
4. Edit storefront details.
5. Add or edit food catalog items.
6. Open the storefront thread to post comments.

## Prototype Limits

- Login, signup, logout, ownership, and subscriptions are simulated.
- Storefronts, food items, and comments are mock or in-memory data.
- The app does not use a real backend, database, auth service, payment service,
  delivery service, or production persistence.
- Location is mocked and uses a simple 2KM radius calculation.
